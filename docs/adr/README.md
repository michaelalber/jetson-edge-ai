# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the jetson-edge-ai project.

## Format

Each ADR is a markdown file named `NNN-short-title.md` with the following structure:

```
# NNN: Title

**Status:** Proposed | Accepted | Deprecated | Superseded by ADR-NNN
**Date:** YYYY-MM-DD

## Context
What situation or problem prompted this decision?

## Decision
What was decided?

## Consequences
What are the trade-offs and downstream effects?
```
