# Scripts

Provisioning, deployment, and utility scripts for the Jetson Orin Nano.

## Planned Scripts

| Script | Purpose |
|---|---|
| `provision-jetson.sh` | Initial Jetson setup: Docker, Ollama, Python tools |
| `deploy.sh` | Copy and start containers on the Jetson over SSH |
| `set-power-mode.sh` | Switch Jetson nvpmodel power profile |

Scripts are shell (bash) and target the Jetson's Ubuntu environment.
